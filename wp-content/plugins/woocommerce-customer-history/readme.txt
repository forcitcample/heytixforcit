Track and store a WooCommerce customer's browsing history with each purchase. Also see their entire purchase history at a glance.

== Description ==

This WooCommerse extension allows you to know every step a user takes prior to completing a purchase on your site. Which pages did they view? Which order did they view them? How long did they spend on each page? What is my customer's actual lifetime value? These are just a few of the questions that you can answer using this plugin.

== Installation ==

1. Upload this plugin using the built-in WordPress plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enjoy viewing customer browsing and purchase history when viewing all new WooCommerce orders.
