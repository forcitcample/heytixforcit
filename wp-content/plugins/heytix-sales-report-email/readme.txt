== Description ==

The Sales Report Emails plugin extends WooCommerce by emailing you a daily, weekly or monthly sales report.

== Installation ==

1. Upload the entire 'woocoommerce-sales-report-email' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
